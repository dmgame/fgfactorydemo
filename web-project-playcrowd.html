Playcrowd

Mobile

Playcrowd is the UK’s first mobile cash treasure hunt game. Interact with the real world and compete against other people.

Play and earn real cash

The application combines mini-games and gambling platform with the ability to win real money. One of the available mini-games is Hunt, in which users travel around the city, look for clues, find prizes and win cash. It is possible to сonnect payment cards, load money to the account and withdraw winnings.

Technology stack
  • Unity3d
  • MapBox - maps and geolocation
  • Spine animations implementation
  • LeenTwean (similar to DOTween, Itween)
  • Rest API
  • Back end sync
  • UniRX
  • Paysafe
  • Payment service implementation
  • In app purchases
