Timber Slash

Casual art

Fgfactory designed game art for characters, backgrounds, icons, UI all other graphical assets for a mobile game.

Tap n slash

Our team has created a completely casual environment and atmosphere for the mobile arcade Timber Slash. The game contains various locations and characters that cut logs and score points. Starting from the first location and the main woodcutter, we created a bright casual style, which was later transferred to other locations and assets.

Promo

To promote the game, we have created a number of game banners for different characters, localizations and different screen orientations of devices.

User interface

Our team created a complete set of in-game screens for the game which included user interface screens and popups, icons for characters and items, other assets and effects. Many elements were animated in Spine and implemented in the Unity engine.
